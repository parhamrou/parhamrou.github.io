Tuberculosis-Detection

Medical

This project implements image classification of chest X-ray images from the Tuberculosis (TB) Chest X-ray Database dataset using transfer learning. The goal of this project is to train a deep learning model that can accurately classify these images as either TB-positive or TB-negative.
